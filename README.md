# CiviX  
### A Crowdsourced Civic Issue Reporting and Resolution System  

CiviX is an AI-powered civic grievance platform designed to simplify how citizens report, track, and verify public issues. It bridges the gap between citizens and authorities by introducing structured reporting, intelligent routing, and transparent resolution workflows.

---

## 📌 Problem Statement 

Urban grievance systems are often:
- Fragmented and slow  
- Lacking transparency  
- Prone to misrouting complaints  
- Difficult for citizens to track  

This results in delayed resolutions and reduced public trust.

---

## 💡 Solution  

CiviX provides a centralized and intelligent platform where users can:
- Report civic issues using **text, voice, or images with GPS**  
- Automatically route complaints using **AI-based classification**  
- Track complaint status in **real-time**  
- View issues on a **public dashboard**  
- Ensure authenticity with **proof-based resolution verification**  

---

## ⚙️ Key Features  

- **Multi-modal Complaint Submission** (Text, Voice, Image + Location)  
- **AI-based Classification (NLP)**  
- **Automated Department Routing**  
- **Real-time Status Tracking**  
- **Public Transparency Dashboard**  
- **Duplicate Detection & Upvoting System**  
- **Privacy & Anonymous Reporting**  
- **Image-based Resolution Verification**

## 🏗️ System Workflow  

1. User submits complaint (text/voice/image + location)  
2. NLP model classifies complaint type  
3. System routes complaint to relevant authority  
4. Authority processes and resolves issue  
5. Proof of resolution is uploaded  
6. System verifies and updates status
